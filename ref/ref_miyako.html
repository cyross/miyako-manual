getOSName
Miyako.getOSName
Miyako(Ruby)が動作している環境をを文字列として返す
現在用意している環境は以下の4種類
"win","linux","mac_osx","other"
書式 Miyako.getOSName #-> String
引数 なし
ブロック ×
戻り値 "win","linux","mac_osx","other"のどれか
アクセス public
例 # 例1
Miyako.getOSName
=>"win" # WindowsXp/Vista/7/...

# 例2
Miyako.getOSName
=>"linux" # Linux

# 例3
Miyako.getOSName
=>"mac_osx" # Mac OS X

# 例4
Miyako.getOSName
=>"other" # *BSD, U*IX, ...
戻る
